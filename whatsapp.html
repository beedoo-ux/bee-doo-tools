🚀 Einrichtungs-Guide
1

Twilio-Account einrichten

Unter twilio.com registrieren. Account SID + Auth Token aus dem Dashboard kopieren und oben eintragen.

2

WhatsApp Business beantragen

In der Twilio Console: Messaging → Senders → WhatsApp. Erste Tests über +1 415 523 8886 (Sandbox). Für Produktion: Facebook Business Verifizierung nötig.

3

Supabase Edge Function deployen

Die Edge Function send-twilio als CORS-Proxy deployen. API-Keys als Supabase Secrets speichern – nie im Frontend!

4

Nachrichtenvorlage genehmigen lassen

WhatsApp Business: Alle Templates müssen von Meta genehmigt werden (~24h). Terminbestätigungen fallen unter UTILITY → günstigste Kategorie.

5

Automatisierung aktivieren

Supabase Cron Job: täglich prüfen welche Termine morgen stattfinden → automatisch WhatsApp senden.

📋 Supabase Edge Function

Deploy-Befehl für die Twilio-Proxy-Function:

// supabase/functions/send-twilio/index.ts
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"
serve(async (req) => {
const { to, message, channel } = await req.json()
const SID = Deno.env.get('TWILIO_SID')
const TOKEN = Deno.env.get('TWILIO_TOKEN')
const FROM = channel === 'whatsapp'
? Deno.env.get('TWILIO_WA_FROM')
: Deno.env.get('TWILIO_SMS_FROM')
const res = await fetch(
`https://api.twilio.com/2010-04-01/Accounts/${SID}/Messages.json`,
{ method: 'POST',
headers: { 'Authorization': 'Basic ' + btoa(`${SID}:${TOKEN}`),
'Content-Type': 'application/x-www-form-urlencoded' },
body: new URLSearchParams({ To: to, From: FROM, Body: message }) }
)
return new Response(JSON.stringify(await res.json()))
})
